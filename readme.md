# Devcamper API
>Backend API for the Devcamper application, which is a bootcamp directory application


## Rename the 'config/config.env.env' to 'config/config.env' and chnage the values to your own

## Install Dependencies
```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production
npm start
```
